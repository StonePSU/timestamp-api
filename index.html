Timestamp Microservice

by Matt Krell

This app is one of the challenges in FreeCodeCamp's backend developer certification course.

This app must meet the following requirements (user stories):

  • Accepts a string parameter and checks to see if the parameters contains either a unix timestamp or a natural language date.
  • If it does, it returns both the unix timestamp AND the natural language form of the date.
  • If the parameter is not a unix timestamp or date then null will be returned for both properties.
